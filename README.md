# الذكاء الاصطناعي للقادة والمدراء

موقع عربي RTL مبني عبر Codex Sites/Vinext لدورة **الذكاء الاصطناعي للقادة والمدراء**. النسخة الحالية هي بوابة أولى قابلة للتطوير للمشاركين، التمارين، الاختبارات والمقاييس، ولوحة تحكم المدرب.

## التشغيل المحلي

```bash
npm install
npm run dev
npm run build
```

في هذه البيئة المحلية قد تحتاج إلى استخدام مسار Node المتاح في Codex:

```bash
PATH=/Users/abozeed/.cache/codex-runtimes/codex-primary-runtime/dependencies/node/bin:$PATH ./node_modules/.bin/vinext build
```

## المسارات

- `/`: الصفحة الرئيسية وتجربة دخول المشارك بالبريد.
- `/about`: تعريف الدورة والفئات المستهدفة ومخرجات التعلم.
- `/exercises`: تمارين أولية قابلة للتوسع مع حفظ الإجابات.
- `/assessments`: اختبارات ومقاييس أولية تدعم اختياراً من متعدد، أسئلة مفتوحة، وتقييماً ذاتياً.
- `/admin`: لوحة تحكم أولية للمدرب مع تصدير CSV.

## البيانات

تم تفعيل D1 عبر `.openai/hosting.json` باسم الربط `DB`. مخطط البيانات موجود في `db/schema.ts`، والهجرة الأولى في `drizzle/0000_mute_saracen.sql`.

الجداول الأساسية:

- `participants`
- `exercises`
- `exercise_answers`
- `assessments`
- `assessment_answers`
- `progress`

أثناء المعاينة المحلية، إذا لم تكن قاعدة D1 مهيأة بعد، يستخدم التطبيق ذاكرة مؤقتة داخل عملية السيرفر حتى تبقى الواجهة قابلة للفحص.

## المحتوى والهوية

- المحتوى المؤقت للتمارين والمقاييس موجود في `app/lib/course-content.ts`.
- أصول الهوية المستخدمة في الموقع موجودة تحت `public/brand/`.
- الخطوط العربية المستخدمة موجودة تحت `public/fonts/`.
